# Taller-De-Estructuras-Lineales
Yobany Andres Nandar Campo;

aplicación de ejercicios de arreglos, Arraylist y Matrices;

Este repositorio contiene el desarrollo de los ejercicios prácticos del taller de Estructuras Lineales. El objetivo es aplicar conceptos de algoritmos, manejo de arreglos y persistencia de datos.

 Contenido del Taller

| Ejercicio | Descripción | Carpeta/Archivo |
| :--- | :--- | :--- |
| **Mínimo y Máximo** | Algoritmo para encontrar los valores extremos en un arreglo. | `src/Arreglos/MinimoMaximo.java` |
| **Frecuencia de Números** | Conteo de apariciones de elementos en un conjunto de datos. | `src/FrecuenciaDeNumeros.java` |
| **Partidos de Liga** | Gestión de encuentros deportivos con lectura de archivos `.txt`. | `src/Arraylist/partidos/` |
| **Matrices** | Operaciones avanzadas (Suma, Transpuesta, Simetría). | `src/Matrices/` |


* **Lenguaje:** Java 
* **IDE:** Visual Studio Code
* **Control de Versiones:** Git & GitHub

Para ejecutar estos ejercicios, asegúrate de tener instalado el JDK de Java y configurar correctamente las rutas de los archivos de texto (como `PartidoLiga.txt`) para evitar errores de lectura.
